# KevinEdu - Lenguaje Corporal: Cómo el Cuerpo Comunica

Esta aplicación web es una guía interactiva sobre el lenguaje corporal, diseñada para enseñar los aspectos fundamentales de la comunicación no verbal.

## Acerca del Proyecto

**KevinEdu** es una plataforma educativa desarrollada por **Kevin Rivera** que explora cómo nuestros gestos, posturas y expresiones faciales transmiten información crucial. El proyecto presenta un temario visual sobre temas clave como:

*   **Kinesia y Gestos**
*   **Expresiones Faciales y Microexpresiones**
*   **Mirada y Contacto Visual**
*   **Proxemia (Espacio Personal)**
*   **Haptica y Tacto**
*   **Postura Corporal**
*   **Gestos de Tensión**
*   **Apariencia Personal**

## Tecnologías Utilizadas

Este proyecto ha sido construido utilizando tecnologías web modernas para asegurar un alto rendimiento y una excelente experiencia visual:

*   **React**: Biblioteca de JavaScript para construir interfaces de usuario interactivas.
*   **Vite**: Herramienta de construcción (build tool) de próxima generación.
*   **Tailwind CSS**: Framework de utilidades CSS para un diseño estilizado y responsivo.
